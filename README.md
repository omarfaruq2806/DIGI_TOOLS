# Digi Tools Hub

A modern React-based web application for exploring and showcasing useful AI tools and digital products in a clean and responsive interface.

## 🛠️ Technologies Used

* React
* Vite
* Tailwind CSS
* React Toastify
* JSON Data

## ✨ Key Features

* Dynamic product cards loaded from JSON data
* Responsive and modern UI design
* Toast notifications for user actions

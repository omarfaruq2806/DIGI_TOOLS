# 🚀 Digi Tools Hub

Digi Tools Hub is a modern and responsive web application built with React that allows users to explore various useful AI tools and digital products in a clean and user-friendly interface.

## 🔗 Live Website
https://digi-tools-hub.netlify.app/

## 📌 Project Overview
This project is designed to showcase digital tools and AI products dynamically using JSON-based data.  
It provides a modern UI, smooth user experience, and responsive layout for all devices.

## 🛠 Technologies Used
- React JS
- Vite
- Tailwind CSS
- JavaScript
- JSON Data
- React Toastify

## ✨ Main Features
- Dynamic product cards loaded from JSON data
- Responsive modern UI design
- Clean card-based layout
- Toast notifications
- Category based tool showcase
- Mobile responsive design

## 📦 Dependencies Used
- react
- react-dom
- react-toastify
- vite
- tailwindcss
